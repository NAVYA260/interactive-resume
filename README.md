### Tech Stack
- React
- React Router
- CSS (mobile-first)

### Description
This is a responsive interactive resume website. Users can log in with hardcoded credentials, view work experience, filter by skills, expand/collapse job details, and download the resume as PDF.

### How to Run
```bash
npm install
npm start
```
